# **App Name**: SIMKONI Web

## Core Features:

- Keyword Search: Search for Islamic terms and notations by keyword, matching against keywords, vocabulary, and categories in a non-case-sensitive manner.
- Contextual Results: Display search results with relevant context, including vocabulary, notation, usage context, and category.
- Detailed View: Provide a detailed view for each term, including full context explanation and usage notes.
- Data Storage: Store vocabulary and notation data in fixed TypeScript files for easy access and management.
- Firebase Hosting: Host the React application using Firebase Hosting.
- Usage Guidance: Provide guidance on how to use the SIMKONI system, including the workflow in cataloging and system limitations.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to create a calm and informative atmosphere.
- Background color: Light gray (#F5F5F5) to ensure readability and a clean interface.
- Accent color: Muted orange (#E59866) to highlight important information and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif, provides a balance of modernity and warmth, making it suitable for both headlines and body text.
- Use simple, clear icons to represent categories and actions.
- Employ a clean and organized layout, emphasizing readability and ease of navigation.
- Use subtle animations for transitions and feedback to enhance user experience without distraction.
